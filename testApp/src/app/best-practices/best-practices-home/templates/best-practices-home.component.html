Best Practices

  • L - Locate code quickly.
  • I - Identify code at a glance.
  • F - Flattest structure possible.
  • T - Try to be DRY.

Immutability - not mutating existing objects in memory, but creating a new one.

using this.object = {{"...this.object, classes: this.object.classes.concat(classId)"}}

Use feature development organization style. Organize your folders/files by features.

If there are more than 7 files in the folder, consider making one or two subfolders.

Have a space in components import section to separate third party imports from your own imports.

Breakdown functions into smaller ones. If the function is more than 10 lines, try to break function into multiple smaller functions.
